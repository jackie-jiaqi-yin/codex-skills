# Chart Rules

Prefer the narrowest figure that answers the current research question.

Order of preference:

1. Reuse a small curated set of existing plots generated by the experiment
2. Generate one compact comparison chart when native plots do not cover the key ranking question
3. Reuse or derive compact result tables from summary artifacts such as `topk`, `summary`, or comparison files
4. Do not create extra visuals if the metric evidence is too sparse or incomparable

Caption rules:

- State what is plotted, the comparison set, and the metric direction.
- Mention the prior best baseline when present.
- Avoid causal claims unless the experiment design clearly supports them.
- Prefer figures that expose actual behavior, such as search traces, loss curves, scatter plots, residuals, confusion matrices, or top-k summary tables.

Do not generate:

- Dense multi-metric dashboards
- Decorative charts with no analytical value
- Figures that hide missing or incomparable runs
