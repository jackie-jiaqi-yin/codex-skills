# Report Composition

Use a stable page shell and adaptive content.

## Always include

- overview cards
- deterministic data quality summary
- the strongest charts
- a grounded narrative block

## Include only when justified

- target-specific sections
- large correlation heatmaps
- distribution deep-dives for many columns
- segment-level commentary

## Good composition rules

- Start with the decision-relevant view of the dataset, not raw detail.
- Surface risks before interesting but low-stakes curiosities.
- Keep the narrative close to the charts it references.
- When a section has no real signal, omit it rather than padding.
- Prefer one strong chart with clear interpretation over three weak charts.
